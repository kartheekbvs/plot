Basic Line Plot

Easy

Create a simple line plot showing the relationship between x and y values. Use matplotlib to plot y = x² for x values from 0 to 10.

Hint

Remember to import matplotlib.pyplot as plt and use plt.plot() to create the line plot. You can create x values using numpy's linspace or range function.

Solution 

# Complete solution
import matplotlib.pyplot as plt
import numpy as np

# Create x values from 0 to 10
x = np.linspace(0, 10, 100)

# Calculate y values (y = x^2)
y = x**2

# Create the plot
plt.figure(figsize=(8, 6))
plt.plot(x, y, 'b-', linewidth=2, label='y = x²')

# Add labels and title
plt.xlabel('x values')
plt.ylabel('y values')
plt.title('Line Plot: y = x²')
plt.grid(True, alpha=0.3)
plt.legend()

# Show the plot
plt.show()
